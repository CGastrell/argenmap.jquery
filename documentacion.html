Inicialización

$().argenmap()

.argenmap() inicializa el plugin principal de ArgenMap. Crea un mapa y prepara el objeto para recibir cualquier otro método (sub-plugins).

Cuando se instancie un mapa con .argenmap(), se creará una instancia de clase ArgenMap y esta quedará almacenada en el objeto jQuery, pudiendo accederse llamando al objeto con el método .data('argenmap').

Para configurar el mapa puede pasarse un objeto como parámetro.

Opciones disponibles
opción tipo valor x defecto descripción
centro array(lat:float,lon:float) [-35.00,-57.00] Establece el centro inicial del mapa
zoom int 4 Establece el zoom inicial del mapa. Los valores van desde 1 a 19
tipo string "baseign" Indica qué capa base debe mostrarse inicialmente. Los tipos disponibles son "baseign", "satelital" y "blanco"
proyeccion string "EPSG:3857" Determina la proyección que se usará para el mapa.
NOTA: la capa satelital sólo puede visualizarse usando la proyección por defecto. Cuando no se use EPSG:3857 se desactivará la capa satelital.
Código
<div id="mapa" style="height:400px"></div>
<div id="mapa2" style="height:400px"></div>
<div id="mapa3" style="height:400px"></div>
<script>
$(document).ready(function(){
  //mapa con todas las opciones por defecto
  $('#mapa').argenmap();
  //satelital
  $('#mapa2').argenmap({tipo:'satelital'});
  //...
  $('#mapa3').argenmap({
	centro:[-33.54,-61.28],
	zoom:9
  });
});
</script>

Vista y manipulación

$().centro(lat,lon)

Centra el mapa en las coordenadas indicadas. Si se llama este método sin parámetros, el método devolverá el centro actual del mapa

Este método rompe la encadenabilidad de jQuery cuando se ejecute para obtener el centro actual del mapa
opción tipo valor x defecto descripción
lat float - Latitud
lon float - Longitud
Código
<div id="mapa" style="height:400px"></div>
<script>
$(document).ready(function(){
  //inicializar el mapa
  $('#mapa').argenmap();

  //establecer nuevo centro
  $('#mapa').centro(-25.57,-60.32);
  
  //pedido del centro actual del mapa, devuelve [-25.57,-60.32]
  $('#mapa').centro();
});
</script>

$().zoom(nivelDeZoom)

Acerca o aleja el mapa al nivel indicado. Si se llama este método sin parámetros, el método devolverá el zoom actual del mapa

Este método rompe la encadenabilidad de jQuery cuando se ejecute para obtener el zoom actual del mapa
opción tipo valor x defecto descripción
nivel de zoom int - Nivel de zoom al que se desea visualizar el mapa. Los valores posibles son entre 1 y 19
Código
<div id="mapa" style="height:400px"></div>
<script>
$(document).ready(function(){
  //inicializar el mapa
  $('#mapa').argenmap();

  //establecer nuevo nivel de zoom
  $('#mapa').zoom(12);
  
  //pedido del nivel de zoom actual del mapa, devuelve 12
  $('#mapa').zoom();
});
</script>

$().capaBase([capa])

Define que capa base utilizar. Si se llama sin parámetros, devuelve la capa base actual del mapa.

Este método rompe la encadenabilidad de jQuery cuando se ejecute para obtener la capa base actual del mapa
parámetro tipo valor x defecto descripción
capa base string - Nombre de la capa a la que se quiere cambiar. Por defecto estas son Base IGN y Satélite
Código
<div id="mapa" style="height:400px"></div>
<script>
$(document).ready(function(){
  //inicializar el mapa
  $('#mapa').argenmap();

  //pedir capa base actual, devuelve 'Base IGN'
  $('#mapa').capaBase();
  
  //cambiar a capa satelital
  $('#mapa').capaBase('Satélite');
});
</script>

$().agregarCapa(opciones)

Agrega una capa al mapa. ArgenMap puede aceptar varios tipos de capas, este método es el más genérico y acepta opciones acorde al tipo de capa a agregar. Para ver las opciones específicas a cada tipo, vea los métodos de conveniencia para cada capa (agregarCapaWMS, agregarCapaKML, etc).

A su vez, este método se utiliza para insertar capas predefinidas. Al momento sólo existen 2: satelital y baseign, pero está previsto agregar otros tipos de capas predefinidas.

Capas predefinidas
opción tipo valor x defecto descripción
capa string - Nombre de la capa predefinida a agregar. Al momento las opciones posibles son "satelital" y "baseign"
Capas genéricas
opción tipo valor x defecto descripción
tipo string - Tipo de capa, "wms" o "kml". Vea agregarCapaWMS() o agregarCapaKML() para entender las opciones de cada una.
... objeto - Vea $().agregarCapaWMS() o $().agregarCapaKML() para entender las opciones de cada una.
Código
<div id="mapa" style="height:400px"></div>
<script>
$(document).ready(function(){
  //inicialización con mapa vacío
  $('#mapa').argenmap({tipo:"blanco"});

  //agregado de capa base IGN
  $('#mapa').agregarCapa('baseIGN');
  
  //agregado de capa satelital IGN
  $('#mapa').agregarCapa('satelital');
});
</script>

$().agregarCapaWMS(opciones)

Agrega una capa desde un servidor WMS al mapa.

Opciones WMS
opción tipo valor x defecto descripción
url string - La URL del servidor WMS donde se pide la capa. Puede ser una dirección IP.
nombre string - Nombre de la capa a agregar. Este es el nombre que aparecerá en el selector de capas. También será la referencia para identificarla con llamadas a $.removerCapa()
formato string image/png Formato MIME de archivo de imagen: "image/jpeg", "image/jpg", "image/png", "image/png8", "image/gif".
transparente boolean true Indica si la capa será transparente o no donde no haya datos para representar.
version string 1.1.1 Versión del servidor WMS para el que se hará el pedido.
esCapaBase boolean false Indica si la capa será tratada como una capa base (las capas base no pueden apagarse, sólo cambiarse por otra capa base).
mostrarAlCargar boolean true Indica si la capa debe ser mostrada una vez que se cargue.
Código
<div id="mapa" style="height:400px"></div>
<script>
$(document).ready(function(){
  //inicializar el mapa
  $('#mapa').argenmap();

  //capa de ríos del IGN
  $('#mapa').agregarCapaWMS({
	url: 'http://www.ign.gob.ar/wms',
	nombre: 'Ríos IGN',
	capas: 'afl',
	formato: 'image/png',
	transparente: true
  });
});
</script>

$().agregarCapaKML(opciones)

Agrega una capa en formato KML al mapa.

Atención: El archivo KML debe estar en el mismo dominio, de lo contrario recibirá un error de Política del mismo origen (Same origin policy).

Opciones WMS
opción tipo valor x defecto descripción
url string - La URL del archivo KML a agregar.
nombre string - Nombre de la capa a agregar. Este es el nombre que aparecerá en el selector de capas. También será la referencia para identificarla con llamadas a $.removerCapa()
Código
<div id="mapa" style="height:400px"></div>
<script>
$(document).ready(function(){
	//inicializar el mapa
  $('#mapa').argenmap();

  //código para agregar capa KML
  $('#mapa').agregarCapaKML({
	url: 'http://dominio.com.ar/puntos.kml',
	nombre: 'Capa de puntos KML'
  });

});
</script>

$().agregarCapaBaseWMS(opciones)

Agrega una capa base desde un servidor WMS al mapa.

Opciones WMS
opción tipo valor x defecto descripción
url string - La URL del servidor WMS donde se pide la capa. Puede ser una dirección IP.
nombre string - Nombre de la capa a agregar. Este es el nombre que aparecerá en el selector de capas. También será la referencia para identificarla con llamadas a $.removerCapa()
formato string image/jpeg "image/jpeg", "image/jpg", "image/png", "image/png8", "image/gif"
version string 1.1.1 Versión del servidor WMS para el que se hará el pedido.
mostrarAlCargar boolean true Indica si debe cambiarse a esta capa base una vez que se cargue.
Código
<div id="mapa" style="height:400px"></div>
<script>
$(document).ready(function(){
  $('#mapa').argenmap().zoom(7);

  $('#mapa').zoom(12);
  
  $('#mapa').zoom();//devuelve 12
});
</script>
