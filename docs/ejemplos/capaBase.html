Vista satelital

En este ejemplo se crea un mapa básico sobre el div con id=mapa utilizando la configuración predeterminada de argenmap y luego se llama a capaBase() pasándole la cadena de texto 'satellite' para mostrar las imágenes de satélite que publica Google con los topónimos reconocidos por la Argentina superpuestos sobre ellas.
