Argenmap

Todos los métodos de la API son plugins de jQuery que trabajan sobre el mapa instanciado en una etiqueta HTML. De tal manera, si instanciamos un mapa en un div: 

<div id="mapa" style="height:300px"></div>
<script>
$('#mapa').argenmap();
</script>
y luego, más adelante en el código o en la estructura de nuestra aplicación, queremos cambiar el centro o agregar un marcador, la llamada será: 
<script>
$('#mapa').centro(-34.1445,-58.84);
$('#mapa').agregarMarcador({nombre:'Mi casa',lat:-34.216,lon:-58.719});
</script>
