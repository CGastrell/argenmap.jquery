Superponer una capa KML sobre el mapa

En este ejemplo se crea un mapa y se lo centra en la ubicación con coordenadas latitud: -34.6476 y longitud -58.4170. Se setea el zoom en el nivel 7. Luego se llama a agregarCapaKML() pasándole un objeto con las propiedades necesarias para hacer referencia al archivo http://sarmientoba.buenosaires.gob.ar/wp-content/themes/plansarmiento/infografia/xml/escuelas.kml publicado por el proyecto SarmientoBA.
