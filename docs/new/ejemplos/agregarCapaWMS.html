Superponer una capa WMS sobre el mapa

En este ejemplo se crea un mapa y se lo centra en la ubicación con coordenadas latitud: -25.9788 y longitud -59.4379. Se setea el zoom en el nivel 7. Luego se llama a agregarCapaWMS() pasándole un objeto con las propiedades necesarias para hacer referencia al servicio WMS de la Infraestructura de Datos Espaciales de Formosa y específicamente a una capa específica dentro de ese servicio WMS.
