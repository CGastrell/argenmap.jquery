1. Guías y tutoriales

Estas guías y tutoriales son ejemplos prácticos del uso de ArgenMap v2. Los mismos están basados en casos de estudio en base a los cuales se concibió ArgenMap. Si buscas ayuda específica de algun método u opción, debes remitirte a la sección de desarrolladores, donde podrás hallar un listado detallado de los métodos y funciones de ArgenMap v2.

ArgenMap es un plugin para jQuery. Todos los ejemplos asumen un conocimiento básico de como se utiliza jQuery. Si no estás familiarizado, te recomendamos que visites el sitio de jQuery para que te enteres rápidamente cómo se utiliza y las ventajas que representa

En estas guías se asume que para ejecutar las pruebas se cuenta con un HTML bien redactado y que lleva incluido el script de jQuery y el de ArgenMap v2 como este: 

<!DOCTYPE html>
<html>
  <head>
  <!-- jQuery, vinculado directo a cdn de google -->
  <script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
  <!-- ArgenMap v2, vinculado directo desde el Instituto Geográfico Nacional -->
  <script type="text/javascript" src="http://mapa.ign.gob.ar/cg/argenmap-v2/build"></script>
  </head>
  <body>
    <div id="prueba" style="height:450px;"></div>
  </body>
</html>

Tal vez tu página no se ve así, pero lo importante es que incluyas los scripts de jQuery y de ArgenMap en el head de tu página: 

<head>
...
<!-- jQuery, vinculado directo a cdn de google -->
<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<!-- ArgenMap v2, vinculado directo desde el Instituto Geográfico Nacional -->
<script type="text/javascript" src="http://mapa.ign.gob.ar/cg/argenmap-v2/build"></script>
...
</head>

...luego colocas un DIV donde quieres mostrar un mapa. Es importante que, sea por CSS o por estilo en el DIV, la altura esté definida. Recuerda asignarle un ID o alguna clase por la cual luego puedas hacer referencia con jQuery

...
<div id="mapa" style="height:400px"></div>
...

Y por último simplemente haces la llamada desde un script en la página. Teniendo en cuenta que las librerías pueden demorar en cargarse, es mejor ejecutar cualquier llamada una vez que la página esté cargada. 

<script>
//con esta llamada, el código se ejecuta
//cuando la página terminó de cargarse
$(document).ready(function(){
  //... líneas de código
  $('#mapa').argenmap();
	//... más líneas de código
});
</script>

Consejo: Siempre es conveniente ejecutar cualquier script luego que la página ha terminado de cargarse. Por este motivo se recomienda que las llamadas se hagan de esta manera para evitar errores difíciles de encontrar.

Código

Todo el código que se encuentra en estas guías se refiere a JavaScript y/o HTML.
A menudo se darán ejemplos con un botón para poder ejecutar el código y ver su resultado.

2. Un mapa

ArgenMap es una librería para mostrar mapas de Argentina. Su función más básica será instanciar un mapa en el/los selector/es indicados con jQuery

.argenmap(opciones)

Las opciones del mapa pueden ser incluidas al momento de inicializarlo mediante un objeto de JavaScript, o bien pueden ser modificadas luego a través de llamadas al mismo selector.

Este es un ejemplo de como inicializar un mapa con distintas opciones:

<script>
//inicialización básica, con centro
//predefinido en latitud -35, longitud -57
$('#prueba').argenmap();

//inicialización con centro en el Aconcagua
$('#prueba').argenmap(
  {
    centro: [-32.6533333,-70.0108333],
    zoom: 12
  });  Ejecutar ejemplo

//vista del Parque Centenario
$('#prueba').argenmap(
  {
    centro: [-34.606942257,-58.435303252],
    zoom: 14
  });  Ejecutar ejemplo
</script>

Resultado

Utilizando un objeto con propiedades centro y zoom se puede configurar el mapa para la zona que se quiere enfocar.

Tipos de mapa

También puede definirse un tipo de mapa. Los valores válidos son baseign y satelital. Por defecto, el tipo de mapa es baseign

Ejemplos

<script>
//inicialización con centro en el Aconcagua
$('#prueba').argenmap({
  tipo: 'satelital',
  centro: [-32.6533333,-70.0108333],
  zoom: 12
});  Ejecutar ejemplo

//vista del Parque Centenario, predeterminado
$('#prueba').argenmap({
  tipo: 'baseign',
  centro: [-34.606942257,-58.435303252],
  zoom: 14
});  Ejecutar ejemplo

//vista del Parque Centenario, satelite
$('#prueba').argenmap({
  tipo: 'satelital',
  centro: [-34.606942257,-58.435303252],
  zoom: 14
});  Ejecutar ejemplo
</script>

Resultado

.centrarMapa(lat, lon, [zoom])

lat (requerido): número indicando los grados decimales de latitud

lon (requerido): número indicando los grados decimales de longitud

lat (opcional): número indicando el nivel de zoom al que se quiere visualizar el mapa.

Si no se provee latitud o longitud, la llamada no tendrá efecto.

Ejemplos

<script>
//se instancia de mapa y
//la llamada puede encadenarse:
$('#prueba').argenmap().centrarMapa(-34.872,-57.965);
Ejecutar ejemplo

//una vez instanciado, podemos simplemente
//hacer la llamada
//vista del Parque Centenario
$('#prueba').centrarMapa(-34.6069,-58.4353,14);
Ejecutar ejemplo

//lago Nahuel Huapi
$('#prueba').centrarMapa(-41.1217,-71.3095,10);
Ejecutar ejemplo
</script>

Resultado

.nivelDeZoom(zoom)

Acerca o aleja la vista del mapa sin cambiar de posición.

zoom (requerido): 1 - 22 número indicando el nivel de zoom al que se quiere visualizar el mapa.

Ejemplos

<script>
//vista del Parque Centenario
$('#prueba').argenmap()
  .centrarMapa(-34.6069,-58.4353,14);
Ejecutar ejemplo

//para alejar, se disminuye el nivel de zoom
$('#prueba').nivelDeZoom(12);
Ejecutar ejemplo

//para acercar, se aumenta el nivel de zoom
$('#prueba').nivelDeZoom(15);
Ejecutar ejemplo
</script>

Resultado

3. Capas

Una vez instanciado el mapa, se puede optar por mostrar capas desde otros servidores o mismo del servicio WMS del Instituto Geográfico Nacional.

.agregarCapa(capa)

Capas predefinidas

Las capas predefinidas son capas que pueden agregarse simplemente por su nombre, sin necesidad de ningún otro parámetro, haciendo la llamada .agregarCapa([nombre])

Base IGN

baseign Es la capa predeterminada, actúa como capa base, pero si el mapa se inicializó como satelital puede luego agregarse esta capa.

Satelital

satelitalInstancia el mapa con una base de imágenes de satélite y los datos del IGN superpuestos.

Advertencia:

No hay límite a la cantidad de veces que puede agregarse una capa, si bien puede resultar divertido, no se recomienda agregar una capa más de una vez.

Ejemplos

<script>
//vista del mapa normal
$('#prueba').argenmap();
Ejecutar ejemplo

//agregar capa satelital al mapa
$('#prueba').agregarCapa("satelital");
Ejecutar ejemplo

//mapa satelital, agregar capa baseign
//encadenando la llamada
$('#prueba').argenmap({tipo:satelital})
  .agregarCapa('baseign');
Ejecutar ejemplo
</script>

Resultado

4. Basic HTML template

With a brief intro into the contents out of the way, we can focus putting Bootstrap to use. To do that, we'll utilize a basic HTML template that includes everything we mentioned in the File structure.

Now, here's a look at a typical HTML file:

<!DOCTYPE html>
<html>
  <head>
    <title>Bootstrap 101 Template</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="http://code.jquery.com/jquery-latest.js"></script>
  </body>
</html>

To make this a Bootstrapped template, just include the appropriate CSS and JS files:

<!DOCTYPE html>
<html>
  <head>
    <title>Bootstrap 101 Template</title>
    <!-- Bootstrap -->
    <link href="css/bootstrap.min.css" rel="stylesheet">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="http://code.jquery.com/jquery-latest.js"></script>
    <script src="js/bootstrap.min.js"></script>
  </body>
</html>

And you're set! With those two files added, you can begin to develop any site or application with Bootstrap.
