Modificando el centro del mapa

En este ejemplo se crea un mapa básico sobre el div con id=mapa utilizando la configuración predeterminada de argenmap y se lo centra en ubicación con coordenadas latitud: -32.65 y longitud -70.01
Ver código Abrir ejemplo 
<script>
//un mapa en una sola línea de código :)
$('#mapa').argenmap();
//cambiamos el centro a las coordenadas del Aconcagua
$('#mapa').centro(-32.6533333,-70.0108333);
Ejecutar ejemplo  

//inicializando el mapa con opciones
var opciones = {centro: [-32.6533333,-70.0108333]};
$('#mapa').argenmap(opciones);
Ejecutar ejemplo  

//encadenando llamadas, en una sola línea de código
$('#mapa').argenmap().centro(-32.6533333,-70.0108333);
Ejecutar ejemplo  
</script>

Modificando el zoom del mapa

En este ejemplo se crea un mapa básico sobre el div con ID mapa utilizando la configuración predeterminada de argenmap y se llama a zoom() para cambiar la escala del mapa al nivel 12 de zoom.
En el ejemplo también se cambia el centro, pero es sólo a los efectos de visualizar un área donde pueda apreciarse mejor el cambio de zoom
Ver código Abrir ejemplo 
<script>
//un mapa en una sola línea de código :)
$('#mapa').argenmap({centro: [-32.653,-70.011]});
//cambiamos el zoom
$('#mapa').zoom(12);
Ejecutar ejemplo  

//inicializando el mapa con opciones
var opciones = {zoom: 12,centro: [-32.653,-70.011]};
$('#mapa').argenmap(opciones);
Ejecutar ejemplo  

//encadenando llamadas, en una sola línea de código
$('#mapa').argenmap().centro(-32.653,-70.011).zoom(12);
Ejecutar ejemplo  
</script>

Mostrando capa satelital

En este ejemplo se crea un mapa básico sobre el div con id=mapa utilizando la configuración predeterminada de argenmap y luego se llama a capaBase() pasándole la cadena de texto 'Satélite' para mostrar las imágenes de satélite que publica Google con los topónimos reconocidos por la Argentina superpuestos sobre ellas.

Notar las mayúsculas y tildes: cuando se configura argenmap para mostrar la capa satélite desde la configuración inicial, será usando {tipo: 'satelite'}, pero cuando se invoca el método capaBase() el parámetro a pasar será Satélite
Ver código Abrir ejemplo 
<script>
//un mapa en una sola línea de código :)
$('#mapa').argenmap({tipo: 'satelite'});
Ejecutar ejemplo  

//inicializando el mapa
$('#mapa').argenmap();
//cambiando a capa base satélite
$('#mapa').capaBase('Satélite');
Ejecutar ejemplo  

//encadenando llamadas, en una sola línea de código
$('#mapa').argenmap().capaBase('Satélite');
Ejecutar ejemplo  
</script>

Un marcador con popup

En este ejemplo se crea un mapa y se lo centra en la ubicación con coordenadas latitud: -32.5642 y longitud -68.6540. Luego se llama a agregarMarcador() sin parámetros que crea un marcador en el centro del mapa.
Ver código Abrir ejemplo 
<script>

  .zoom(10);

  lon: -68.83364868164072,
  contenido: '<h3>Hola mundo!</h3>Soy un marcador con contenido!',
  nombre: 'marcador1'
});

</script>

Marcadores agrupados

En este ejemplo muestra como agrupar marcadores definiendo la propiedad capa del marcador. Cuando se define capa argenmap busca una capa con ese nombre y, de no existir, la crea. Luego indicamos si la capa debe listarse en el administrador de capas con listarCapa en true.
Hacé click en el administrador de capas para ver las capas listadas.
Ver código Abrir ejemplo 
<script>
//un mapa en una sola línea de código :)
$('#mapa').argenmap();

//tomamos los marcadores del ejemplo anterior, pero agregamos
//capa y listarCapa para poder ver
//los grupos en el administrador de capas
var marcadores = [
  {nombre: 'cataratas', lat: -25.695278, lon: -54.436667, contenido:'Cataratas del Iguazú!', capa: 'Maravillas', listarCapa:true},
  {nombre: 'atucha', lat: -34.05, lon: -59.2, contenido:'Atucha: central eléctrica', capa: 'Recursos', listarCapa: true},
  {nombre: 'minera', lat: -27.31,  lon: -66.608617, contenido:'Minera Lumbrera: buena mina', capa: 'Recursos'},
  {nombre: 'vacio', lat: -32.911,lon: -68.833, contenido:'No soy solo un marcador vacío, tengo sentimientos :(', icono:'hueco', capa: 'Maravillas'}
];
$('#mapa').agregarMarcadores(marcadores);
Ejecutar ejemplo  
</script>

Superponiendo capas WMS

En este ejemplo se crea un mapa y se lo centra en las coordenadas latitud: -25.9788 y longitud -59.4379. Se setea el zoom en el nivel 7. Luego se llama a agregarCapaWMS() pasándole un objeto con las propiedades necesarias para hacer referencia al servicio WMS de la Infraestructura de Datos Espaciales de Formosa y específicamente a una capa dentro de ese servicio WMS.
Ver código Abrir ejemplo 
<script>
//un mapa en una sola línea de código :)
$('#mapa').centro(-25.9788, -59.4379).zoom(7);

//configuración para una capa del WMS de Formosa
var formosa = {
  url:'http://idef.formosa.gob.ar/servicios/wms?',
  capas:'mce_prov_escuelas',
  nombre: 'Escuelas de la provincia de Formosa'
};
$('#mapa').agregarCapaWMS( formosa );
Ejecutar ejemplo  
</script>

Superponiendo tus datos con KML

En este ejemplo se crea un mapa y se lo centra en la ubicación con coordenadas latitud: -34.92 y longitud -57.92. Se setea el zoom en el nivel 12. Luego se llama a agregarCapaKML() pasándole objetos con las propiedades necesarias para hacer referencia a 2 archivos KML.
Ver código Abrir ejemplo 
<script>
//un mapa en una sola línea de código :)
$('#mapa').centro(-25.9788, -59.4379).zoom(7);

//configuración para una capa del WMS de Formosa
var arroyos = {
  url:'arroyos.kml',
  nombre: 'Cuencas de arroyos'
};
var inundacion = {
  url:'inundacion.kml',
  nombre: 'Zonas inundadas'
};
var topografia = {
  url:'topografia.kml',
  nombre: 'Curvas de nivel'
};
$('#mapa').agregarCapaKML( inundacion )
  .agregarCapaKML(arroyos)
  .agregarCapaKML(topografia);
Ejecutar ejemplo  
</script>
